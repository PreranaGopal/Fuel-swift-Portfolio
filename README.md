# FuelSwift Portfolio
A full-stack Fuel Delivery & Slot Booking application.

## Features
- User-friendly slot booking
- Real-time fuel delivery tracking
- Clean UI/UX design

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, MongoDB
- Figma for UI/UX

## How to Run
1. Clone the repository
2. Install dependencies: `npm install`
3. Run the app: `npm start`
